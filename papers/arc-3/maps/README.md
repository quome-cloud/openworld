# ARC-AGI-3 world-model cards (the atlas)

Every ARC-AGI-3 solve becomes a serveable OpenWorld `World`; `render_card` draws it as a
self-contained SVG **model card**. This folder holds those cards, segmented by solve type.

## The canonical atlas — the Claude Fable 25/25 source-free solves

`<game>.svg` (25 files, one per public game) are the headline solves from the **Claude Fable 5**
source-free **25/25** run: each card shows the discovered `start → level 1 → … → win`
state-transition graph, the masked-frame state schema, the actions, the verified learned-table
dynamics, and the reward (`levels_completed`). Step counts match the banked Fable solutions
(e.g. `bp35` 384, `lf52` 914, `su15` 141). These are the source of the paper's atlas gallery
figure (`figs/arc3_maps_gallery.png`) and are built by `scripts/build_arc3_fable_maps.py`.

`world_model_forming.{svg,pdf,json}` — the "forming vs verifiable world model" figure
(`scripts/make_forming_figure.py`).

## `opus/` and `codex/` — the other two source-free arms

The same atlas cards for the two weaker arms, one card per game **each arm fully solved**
(so a game appears only where that arm actually reached the win):

- `opus/<game>.svg` — the primary **Claude Opus 4.8** arm (**16** cards). Same games as the
  Fable atlas minus the walls Opus did not clear; step counts are Opus's own (e.g. `su15` 105 vs
  Fable's 141).
- `codex/<game>.svg` — the **GPT-5.5 (Codex)** arm (**12** cards).

Rendered by `scripts/build_arc3_arm_maps.py {opus,codex}` from
`experiments/results/arc3_fullgame_sourcefree{,_codex}.json` — the identical recipe as the Fable
atlas, pointed at each arm's archive. The paper's gallery figure still uses only the top-level
Fable cards; these subfolders are for browsing the model-scaling story arm-by-arm.

## `multiworld/` — alternative modelings (exploratory, NOT the headline solves)

The `_online` / `_regimes` / `_resynth` / `_consensus` cards are **not** source-free solves. They
are alternative world-model renderings from the **E120–E123 multi-world experiments** — online
causal monitoring, surprise-triggered regime reset, self-resynthesis, and expert-consensus voting
— which were removed from the paper. They are retained here as artifacts and are regenerated by
`experiments/e12{0,1,2,3}_*.py` (now writing under `multiworld/`). Nothing in the paper depends
on them.
